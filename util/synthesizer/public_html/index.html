Synthesizer

Results Here is where you find the results of all benchmarks.
Start a batch If you want the synthesizer to do a benchmark right now, this is where to go.
Check if running Before starting a batch manually, you should check if a batch is running. You can do that here, and also kill batches in progress, and clean up the temporary directory.
Update designs Do a svn update in the designs directory to update the configurations for the designs. The location of the designs in the repository is bs/bsc/trunk/util/synthesizer/DESIGNS. To update the scripts or the web pages of the synthesizer, log in and do a svn update manually.
Documentation Adding designs, adjusting design parameters, brief overview of the system (link goes to websvn, the file is in bs/bsc/trunk/util/synthesizer/description.txt)
